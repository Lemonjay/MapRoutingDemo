# MapRoutingDemo
一个基于pGrouting实现最短路径规划的示例，使用到的开源工具geoserver、postgressql、postgis、pGrouting，pgsql的数据访问使用Npgsql 

根据起止点坐标找到起止导航的坐标点，虚线为无路网导航，红色线为基于路网的导航
